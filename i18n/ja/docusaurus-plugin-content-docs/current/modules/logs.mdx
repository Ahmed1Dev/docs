# ログ

ProBotがすべてのサーバーアクションを即座に記録します

:::info

ここで説明されている設定はダッシュボードで管理されます。

1. ダッシュボードにアクセスしてログインします。
2. 左側のメニューからサーバーを選択します。
3. モジュールを選択し、有効になっていることを確認します。

:::

## 設定

各ログにはイベント名、ログチャンネル、および埋め込みカラーが含まれます。

:::important

一部のログはパフォーマンス上の理由から、ティア2のプレミアムストレージが必要になる場合があります。

:::

- メンバーがBANされました


- タイムアウト


- チャンネルが作成されました


- ロールが作成されました


- チャンネルが削除されました


- メッセージが削除されました


- ロールが削除されました


- メッセージが編集されました


- メンバーがキックされました


- メンバーが別のボイスチャンネルに移動されました


- メンバーがボイスチャンネルから切断されました


- メンバーが参加しました


- メンバーが退出しました


- ニックネームが変更されました


- モデレーションコマンドが使用されました


- ロールが付与されました


- ロールが削除されました


- サーバーの招待


- メンバーのBANが解除されました


- チャンネルが更新されました


- チャンネルの権限が更新されました


- ロールが更新されました


- サーバーが更新されました


- メンバーがボイスチャンネルに参加しました


- メンバーがボイスチャンネルを退出しました


- ボイスステータス


- メンバーがボイスチャンネル間を移動しました


:::caution

時々、ボットが権限の問題でログにアクションを記録しないことがあります。その場合は、ボットをキックして再招待し、`Manage webhooks`の権限があることを確認してください。

:::
